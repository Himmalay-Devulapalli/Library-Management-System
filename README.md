# Library-Management-System

 Library Management system with attractive graphical user inteface developed using python oops concept, Tkinter, Fireabase.
 
# Features
 Splash Screen\
 Main Menu\
 QR Code Scan\
 Issue Book\
 Return Book\
 Renew Book\
 Studen History\
 Notify Delay\
 Admin Access
